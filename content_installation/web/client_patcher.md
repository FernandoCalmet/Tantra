# Client Patcher

To make an update from the Client, we must first make sure that we have the files with the following structure on our website.

```note
patch/
  |-- main/
        |
        -- PatchFold/
        |   |
        |   -- ``Here my updated files``
        |
        -- mainupdate/
            |
            -- Patcher.dat
            |
            -- Tantra.dat
            |
            -- Tantra.exe
            |
            -- UpdateList.dat
            |
            -- Version.dat
```
